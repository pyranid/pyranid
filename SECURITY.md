# Security Policy

## Reporting a Vulnerability

Please report suspected vulnerabilities privately by emailing security@revetware.com.

Include the affected Pyranid version, a concise description of the issue, and any reproduction steps or proof-of-concept details that can be shared safely. Please do not open a public GitHub issue for suspected vulnerabilities until we have coordinated disclosure.

## Supported Versions

Security fixes are prioritized for the latest released version of `com.pyranid:pyranid`. Snapshot builds are development artifacts and may change without notice.
